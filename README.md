## CHANGELOG

### （软件）20190529 第一个版本

- 支持物联网开发、支持定制积木插件、支持云端托管、支持简体和繁体等多语言。
- 提供了大量基础教育场景范例、支持分享和转发、载入与保存等基本功能。
- 支持 Windows 离线应用、也支持二次 HTML5（Webcomponents） 、 JavaScript 、 NodeJS 云端开发代码。
